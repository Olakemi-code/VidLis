RESUME AUTOMATIQUE DE VIDEO IA

C'est un outil générateur de rapport structuré à partir d'une vidéo Toutube portant sur un sujet d'IA. Il extrait et résume le contenu de la vidéo Youtube ciblée. Le résumé obtenu comporte :
- les concepts clés;
- les outils, librairies mentionnés;
- les applications ou cas d'usage;
- et un résumé technique final.

Une fois le lien inscrit, l'utilisateur a la possibilité de lire la vidéo directement sur l'application. Il a le choix de la langue (Français et Anglais), du format (résumé court ou détaillé), peut télécharger le rapport généré, ou même l'effacer afin de procéder à une nouvelle analyse. 

Stack Technique 
- Python
- IA générative : modèle LLM (Gemini 3 Flash), clé API de Google Gemini
- yt-dlp
- Prompt Engineering
